# 問題
## [ABC 014 C - AtColor](https://atcoder.jp/contests/abc014/tasks/abc014_3)

```ruby
n = gets.to_i
s = Array.new(1000000+10, 0)

n.times do
  a, b = gets.split.map(&:to_i)
  s[a] += 1
  s[b+1] -= 1
end
(1..1000000).each do |i|
  s[i] += s[i-1]
end

puts s.max
```
