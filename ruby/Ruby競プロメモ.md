### map

### inject

- 数列の和を計算したいときに使う
- Ruby 2.x からはsumメソッド

#### 例

```ruby
# 数列の和を計算
irb(main):003:0> [1,3,5].inject(:+)
=> 9

# 初期値を指定したい場合
irb(main):009:0> [1,3,5].inject(100, :+)
=> 109


```
